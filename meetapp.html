Meet App

The project is a web-based app for managing and discovering events. It allows users to search for events by city, filter results, and view details of upcoming events. The main objective is to help users stay informed about events happening in various locations, utilizing features like event search, genre-based charts, and maps. The site is built with React and integrates APIs for event data.

Technologies used:

  • HTML
  • CSS
  • React
  • GoogleAuth
  • GoogleCalendarAPI
